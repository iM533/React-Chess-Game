# React Chess Game ♟️
Interactive chess built with React and TypeScript: correct piece movement rules, move highlighting, and player switching. Deploys in one click via Vercel.

## Demo
- **Live**: https://react-chess-game-9uarmafu3-im533-3421s-projects.vercel.app/

## Features
- Correct move logic for all pieces with validation
- Highlighting of legal moves
- Current-player indicator and turn switching
- Minimal, responsive UI
- Fast build with CRA + React 19 + TypeScript

## Tech Stack
- React 19, TypeScript
- CRA (react-scripts), npm

## Quick Start (local)
bash npm install npm start
The app runs at `http://localhost:3000`.

## Scripts
- `npm start` — dev server
- `npm run build` — production build

## Project Structure (brief)
- `src/models` — board, cells, and pieces logic
- `src/components` — UI components for board and cells
- `src/images` — sprites/icons
- `src/App.tsx` — UI composition entry point
